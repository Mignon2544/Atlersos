Q&A Answer Key

1
What type of annotation is required for this project - granular or coarse?
Correct Answer:
Coarse - focus on main actions, minute details are not required
2
What is the primary focus of annotations in this project?
Correct Answer:
Main actions and hand dexterity relevant to the task
3
If a label says "pick up book" without specifying which hand was used, is this acceptable?
Correct Answer:
Yes - this is a completely valid annotation
4
Are labels using "lift", "pick up", and "grab" interchangeably acceptable?
Correct Answer:
No - "Pick up" and "grab" are not synonymous, and you should use consistent language in the same episode
5
How should labels be written?
Correct Answer:
As direct instructions (e.g., "pick up the cup")
6
The ego is lifting a shirt and shaking it out. Which labels are acceptable?
Correct Answer:
"Straighten the shirt", "Pick up the shirt and move it back and forth", or "Lift the shirt and shake it out" - all acceptable
7
The ego picks up a mat from the floor and places it on the table. Which label is acceptable?
Correct Answer:
"Pick up mat and place on table" or "Move mat to table" - must capture both actions
8
Can you use relative positions like "left" and "right" in labels?
Correct Answer:
Yes - as long as they are accurate from the ego's point of view
9
If one annotator calls an object "orange box" and another calls it "yellow box", is this a problem?
Correct Answer:
No - inconsistent naming is acceptable unless clearly incorrect (e.g., calling orange "blue")
10
What distinguishes a MAJOR action (must be labeled) from a MINOR action (doesn't need labeling)?
Correct Answer:
Major actions directly contribute to the task goal; minor actions are incidental or transitional
11
A video contains PII (Personally Identifiable Information) such as a person's face. What should you do?
Correct Answer:
Flag as "Contains PII", do NOT annotate the PII segment, but QC the rest of the task normally
12
What is required for timestamps to be correct?
Correct Answer:
Begin timestamp: when hands start reaching toward object. End timestamp: when hands disengage. Must be accurate
13
When multiple events happen at once, what must the labels capture?
Correct Answer:
All major actions contributing to the goal
or
All actions with meaningful object interactions
14
Is it acceptable to use sequence words like "another" and "next" for repeated actions?
Correct Answer:
Yes - as long as each action and label is accurate for the given timestamp
15
If a label describes one action in a segment that contains multiple actions, is this acceptable?
Correct Answer:
Yes - but only if all MAJOR actions are captured; minor/incidental actions don't need labeling
16
When should a label be "idle" or "no action"?
Correct Answer:
Only when both hands are still or not interacting with objects
17
The ego is walking to move dishes from one table to another. How should this walking be labeled?
Correct Answer:
Label as part of the task - walking is necessary and related to the task
18
A label says "pick up weight and frame" but only the weight was touched. What should happen?
Correct Answer:
Reject - this is a half-true label / hallucination
or
Edit to "pick up weight
19
The ego's movement suggests they're about to put down a phone, but they never actually do. Can the label say "put down the phone"?
Correct Answer:
No - label only what can be seen actually happening in the video
20
An action briefly leaves the camera's frame (e.g., placing item on top shelf where only lower shelf is visible). Is this acceptable?
Correct Answer:
Yes - if the action is clearly happening and can be confirmed from on-screen information
21
Should timestamps that are off by a few seconds (within tolerance) be flagged as incorrect?
Correct Answer:
No - only large/clearly wrong timestamp mistakes should be flagged
22
The subject picks up a red mug from the sink while a blue mug sits on the counter in-frame. Which label is correct?
Correct Answer:
"pick up mug from sink" or "pick up red mug"
23
The subject washes their hands, checks their watch, and adjusts their head-mounted camera. How should this be labeled?
Correct Answer:
"no action"
24
The subject slightly adjusts the corners of a t-shirt to see the design more clearly. Which label is appropriate?
Correct Answer:
"adjust t-shirt" or "straighten t-shirt"
25
What is the maximum word count for a single dense label segment?
Correct Answer:
"20 words (about 4 atomic actions)"
26
Can you combine "No Action" with other actions in a single label?
Correct Answer:
No - never combine No Action with real actions in the same label
